# 📖 Overview

The **Spam Mail Detector** is a machine learning and Natural Language Processing (NLP) project that classifies incoming email or SMS messages as **Spam** or **Ham (Not Spam)**.

The project converts text into numerical features and uses a classification model to predict whether a message is legitimate or spam.

---

# 🎯 Objective

Develop an intelligent spam detection model capable of automatically classifying messages with high accuracy.

---

- Source: UCI Machine Learning Repository
